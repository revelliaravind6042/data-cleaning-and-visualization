# 📊 Data Cleaning & Visualization Project

## 📌 Project Overview

This project focuses on cleaning, preprocessing, analyzing, and visualizing a raw student performance dataset using Python.

The objective is to transform raw data into meaningful insights through data cleaning techniques and visual representations.

---

## 🎯 Objectives

- Clean and preprocess raw data
- Handle missing values and duplicates
- Perform exploratory data analysis (EDA)
- Create meaningful visualizations
- Extract insights from the dataset

---

## 🛠️ Tools & Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📂 Dataset

**Dataset:** Student Performance Dataset

The dataset contains student academic information such as:

- Math Scores
- Reading Scores
- Writing Scores
- Student Performance Metrics

---

## 🔍 Data Cleaning Tasks Performed

### Data Preprocessing

- Loaded dataset
- Checked missing values
- Removed duplicate records
- Verified data quality
- Prepared data for analysis

### Data Analysis

- Statistical Summary
- Correlation Analysis
- Feature Exploration

---

## 📊 Visualizations

### Correlation Heatmap
Visualizes relationships between numerical features.

### Score Distribution Analysis
Shows how scores are distributed across students.

### Performance Dashboard
Provides a visual summary of key insights.

---

## 📈 Key Insights

- Strong positive correlation exists between reading and writing scores.
- Student performance patterns can be identified through visual analysis.
- Data visualization helps uncover hidden trends and relationships.
